Conhecimentos Adquiridos no Curso de HTML e CSS

Durante o curso de HTML e CSS oferecido pela Alura, tive a oportunidade de explorar diversos conceitos fundamentais para o desenvolvimento web. Abaixo, apresento um resumo dos principais tópicos que aprendi:

Primeiros Passos com HTML

Comecei entendendo a diferença entre criar um documento no Google Docs e desenvolver uma página em HTML. Enquanto o Google Docs é focado em edição de texto visual, o HTML exige a escrita de código para estruturar o conteúdo. Para isso, aprendi a instalar e configurar o Visual Studio Code, um editor de código leve e poderoso, além de criar e organizar pastas para os projetos.

Entendendo o HTML

Aprendi que o HTML (HyperText Markup Language) é uma linguagem de marcação, utilizada para estruturar o conteúdo de páginas web. Compreendi a importância da documentação oficial do W3Schools (W3S) como fonte confiável de consulta.
Com base nisso, explorei a estrutura básica de um documento HTML, entendendo o papel da declaração <!DOCTYPE html> e a organização entre as seções <head> (onde ficam as metainformações) e <body> (onde está o conteúdo visível da página). Também aprendi sobre o uso de tags semânticas, que melhoram a acessibilidade e a organização do código.
Além disso, conheci o conceito de "quirks mode" (modo de compatibilidade) e como evitá-lo garantindo a correta renderização da página.

Acessibilidade e Ferramentas

Acessibilidade foi um dos temas abordados, e aprendi a criar textos alternativos (atributo alt) para imagens, garantindo que leitores de tela possam interpretar o conteúdo. Também utilizei as Developer Tools dos navegadores para inspecionar e depurar elementos da página.
O uso de extensões como o Live Server foi essencial para visualizar as alterações em tempo real, facilitando o desenvolvimento e testes.

Figma e Layouts

Entendi como consultar layouts de projetos no Figma, ferramenta de design muito utilizada por designers e desenvolvedores. Através dela, aprendi a extrair cores, medir espaçamentos e seguir um guia visual para replicar a interface com fidelidade no código.

Introdução ao CSS

Avançando para o CSS (Cascading Style Sheets), aprendi a aplicar estilos aos elementos HTML. Descobri como criar arquivos externos de CSS, vinculando-os ao HTML para uma melhor organização do projeto.
Explorei as principais propriedades CSS, como cor, espaçamento, tamanhos e fontes, além de aprender como aplicar cores no CSS, seja usando nomes de cores, códigos hexadecimais ou paletas externas. Também pratiquei alterar cores de fundo, do texto e destacar partes do conteúdo, melhorando a aparência geral da página.
